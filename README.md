 # tsub

**tsub** a game made with Raylib and C3 to test your tapping and tracking skills.

## Motivation

The motivation for this project was to learn the [C3](https://c3-lang.org) language and the [Raylib](https://raylib.com) library.


## Run

**NOTE**: This project requires the [C3 compiler](https://github.com/c3lang/c3c).

1. Clone the repository:
```
git clone https://github.com/vishavish/tsub.git
```

2. Run the project:
```
cd tsub
c3c run
```
