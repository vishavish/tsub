 # tsub

**tsub** is a game written in C3 and Raylib.

## Purpose

The motivation for this project is to learn the [C3 language](https://c3-lang.org/) and [Raylib library](https://www.raylib.com/).

## Run

**NOTE**: This project requires the [C3 compiler](https://github.com/c3lang/c3c).

1. Clone the repository:
```
git clone https://github.com/vishavish/tsub.git
```

2. Execute the following commands:
```
cd tsub
c3c run .
```
